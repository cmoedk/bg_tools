# 4_playtest

Games that are ready for **playtesting with other people**.

## Contents

Each game lives in a folder named `[working_title]` and contains the same files
as [`3_test`](../3_test/), with one change:

- `[working_title]_rules_0.x.playtest.md` — the versioned rules file.

### Notes

After a session, add the playtest to the relevant date header in the notes file,
e.g. `## 4. June 2025 — Playtested with 4 players (name1, name2, name3, name4)`.

### Changelog

Versions now use the format `## Version x.y (23 May 2025)`. Increment `y` for
each date; increment `x` for major rules changes.

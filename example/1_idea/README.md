# 1_idea

Games that exist only as a **concept or idea**.

## Contents

Each idea is a single file. There is no required structure to the file — just
capture the concept.

- `[working_title].idea.md`

The **working title** must be meaningful and in English.

## Moving up to 2_draft

When an idea is ready for draft rules:

1. Create a folder in [`../2_draft/`](../2_draft/) named `[working_title]`.
2. Rename `[working_title].idea.md` to `[working_title].notes.md` and move it
   into the new folder.
3. Create a `[working_title]_rules.draft.md` file in the new folder.

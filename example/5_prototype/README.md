# 5_prototype

Games that are ready to be **tested by other people** and to be presented at
events or to a publisher.

## Contents

Each game lives in a folder named `[working_title]` and contains the same files
as [`4_playtest`](../4_playtest/), with these additions/changes:

- `[working_title]_rules_0.x.prototype.md` — the versioned rules file.
- `[working_title].info.md` — game information used for marketing, such as:
  - Release title
  - Player count
  - Player age
  - Game type
  - Game mechanisms
  - Estimated game time
  - Pitch
  - Why does this game exist?
  - Why is this game interesting for a publisher?

### Notes

Append feedback to the relevant date header when applicable, e.g. comments from a
publisher, a game event, or another expert.
